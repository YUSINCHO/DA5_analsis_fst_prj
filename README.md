# 코드잇 스프린트 DA5_ 초급 프로젝트
## 따릉이 고장 예방 및 관리 방안 제안 프로젝트

### 프로젝트 목적
1. 데이터 전처리 학습
   데이터 분석을 위한 공공데이터 수집과 데이터 EDA 및 새로운 데이터셋 생성하고 이를 기반으로 분석하는 모든 단계를 직접 구현
2. 통계적 검정 학습
   전처리를 끝낸 공공데이터을 통해 가설을 선정 및 기술 검정, 통계 검정을 하여 각 변수별 상관관계를 알아보는 과정을 학습

### 프로젝트 개요  
이 프로젝트는 공공자전거 따릉이의 고장 관리가 원할하지 않아 서울시민의 세금을 투입하여 적자 운영 중인 따릉이 사업 이용자 수가 점차 줄어드는 형상을 기사로 접하고 따릉이 고장 원인의 상관관계를 알아보고자 한다.

`PART 1) 과제 개요`
- 문제 상황
  - 따릉이 관리 인력 부족 + 따릉이 고장에 의한 이용자 유입 감소
- 분석 목적
   - 따릉이 고장 예방 및 관리 효율화
- 담당 분석 요인
   - 따릉이 고장 데이터 분석으로 따릉이 고장과 상관 관계가 있는 내부/외부 요인 확인
   - 외부적 요인인 날씨에 따른 고장 분석
      - 중점 분석 대상 : 날씨 변수 중 기온과 타이어 고장 간의 관계

`PART 2) 데이터 분석 및 EDA`

__날씨 데이터_고장 분석__
  
`기온 관련`
- 모든 고장 부품들은 기온과 상관관계가 양의 상관관계를 가지고 있음을 확인할 수 있음

`강수 및 적설 관련`
- 강수량과 고장 간의 상관관계는 전반적으로 낮음 (-0.1 ~ 0.3)
- 적설량은 대부분의 부품과 음의 상관관계를 보임 (-0.2 ~ -0.5)

`습도 관련`
- 평균 상대습도는 고장 유형과 미미한 상관관계를 보임


3_2. 강수 관련 데이터 제외 

`높은 결측치 비율`
- 일별 연 단위 데이터셋은 365 행으로 구성, 이 데이터 중 상당수가 결측치로 확인
   - 일강수량 : 215일 결측 → 약 58%의 데이터 손실
   - 일최심신적설 : 356일 결츨 → 약 97%의 데이터 손실
   - 일최심적설 : 349일 결측 → 약 96% 의 데이터 손실
 
`데이터 처리의 한계`
- 미세 강수량의 경우, 결측치가 아닌 0으로 기록되어 데이터 보정이 어려움
   - 일반적인 결측치 처리 방법 적용이 부적절함을 판단
 
3_3. 날씨 데이터 분석 방향성 설정

`최종 분석 범위`
- 신뢰성 있는 기온 관련 변수(평균기온, 최고기온, 최저기온, 평균지면온도)에 초점을 맞
춘 분석 진행
- 강수 관련 변수들은 데이터 품질 문제로 인해 분석에서 제외
- 타이어 고장과 기온 간의 관계를 중심으로 한 예방적 관리 방안 도출에 집중

3_4. 설정된 분석 방향성에 따른 고장 유형 재분석

`고장 빈도 측면`
- 단말기, 페달, 안장, 체인 등 다른 부품과 비교했을 때, 타이어의 경우 기타를 제외한 고
장 유형 중 가장 높은 고장 빈도를 나타냄

`기온과의 상관관계`
- 2021년 ~ 2023년의 3개년 데이터에서 타이어는 기온 관련 변수들과 일관되게 높은 상
관관계를 보임
- 평균기온, 최고기온, 최저기온, 평균지면온도와의 상관계수가 타 부품 대비 높음
- 연도별로 상관관계가 일관되어 높은 신뢰성을 보임

3_5. 날씨 데이터와 고장 유형 분석 방향

`기온-타이어 관계성 분석` 
- 기온 변화에 따른 타이어 고장 패턴 심층 분석
- 월별(계절별?) 타이어 고장 특성 파악
- 기온 변수별(평균기온, 최고기온, 최저기온, 평균지면온도) 영향력 차이 분석

`예방적 관리 전략 수립`
- 기온 변화에 따른 타이어 관리 방안 도출
- 온도별 차별화된 점검 주기 설정?

4_1. 월별 타이어 고장 패턴 분석
`연도별 일관된 패턴`
- 2021년 ~ 2023년간 타이어 고장 건수가 매우 유사한 패턴을 보임
- 8 ~ 10월에 최고점을 기록하고, 1~2월에 최저점을 기록함

`계절적 특성`
- 여름철 (7 ~ 9월) : 고장 건수가 급격히 증가
- 겨울철 (12 ~ 2월) : 고장 건수가 현저히 감소

5_1. 결론

- 최고기온이 높아질수록 타이어 고장 건수가 증가하는 양의 상관관계가 나타났습니다.    
- 하지만 33°C 이상 폭염일에는 이용률 감소로 인해 타이어 고장 건수가 감소하는 경향이
일부 나타났습니다.

5_2. 시사점 
- 전반적으로 기온이 상승할수록 타이어 고장 건수가 증가하는 경향이 확인되었습니다.
- 폭염일의 고장 건수 감소는 폭염일떄 따릉이 이용량 감소와 관련이 높을 것으로 보입니다.
